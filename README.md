# ChatFilterIndo
Tujuannya untuk belajar, yaitu untuk membuat aplikasi yang bisa menyensor kata kasar.

Yang sudah diperbaiki:
- Kata yang memiliki karakter unik.
- Teks yang memiliki kalimat kasar berulang.
